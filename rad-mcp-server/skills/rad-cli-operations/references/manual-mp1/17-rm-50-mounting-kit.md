Mounting Kit 
This document describes the installation of 19-inch units in 19-inch and 23-inch racks/frames using the following mounting kits:  
• 
RM-50/19 for installing a 19-inch device in a 19-inch rack (flat installation; see Figure 1) 
• 
RM-50/19/A for installing a 19-inch device in a 19-inch rack, both deep (100 mm) and flat installation of the device (see Figure 2 and Figure 3). 
This kit replaces the former RM-53. 
• 
RM-50/23 for installing a 19-inch device in a 23-inch rack (see Figure 4) 
• 
RM-50/23F for installing a 19-inch device in a 23-inch frame (see Figure 5)  
Note: Use these kits for mounting 19-inch devices with RM-50 mentioned in their data sheets. 
Note: For installing a 19-inch device on a wall, use the WM-50 mounting kit.

INSTALLATION  
 
Warning 
Disconnect all cables including the power cord from the unit before 
performing the mounting procedures. 
Installing a 19-inch device in a 19-inch rack  
You will need: 
• RM-50/19 kit 
• Philips Type 1 screwdriver (not provided) 
• Four M5 or M6 screws (not provided) 
 
To install a 19-inch device in a 19-inch rack: 
1. 
Fasten one of the brackets to the left side of the unit using four 6-32 UNC screws as shown in the figure.  
2. 
Repeat step 1 to fasten the second bracket to the right side of the unit. 
3. 
Place the unit in the rack and fasten the brackets to the side rails of the rack using two M5 or M6 Philips screws (not included 
in the kit) on each side.

Installing a 19-inch device in a 23-inch rack  
You will need: 
• RM-50/23 kit  
• Philips Type 1 screwdriver (not provided) 
• Eight M5 or M6 screws (not provided) 
 
To install a 19-inch device in a 23-inch rack: 
1. 
Fasten one of the short front brackets to the left side of the unit using four 6-32 UNC screws as shown in the figure. 
2. 
Fasten one of the short rear brackets to the left side of the unit using four 6-32 UNC screws. 
3. 
Fasten one of the long brackets to the rear frame of the rack using two M5 or M6 Philips screws (not included in the kit). 
4. 
Repeat steps 1-3 on the right side of the unit. 
5. 
Place the unit so that the long brackets on each side fit in the hole of the rear short brackets.  
6. 
Fasten the brackets to the side rails of the rack using two M5 or M6 Philips screws on each side (not included in the kit).
