KIT CONTENTS 
Rack adapter components for installing a single unit in a 19”/23” rack include: 
• One bracket 
 
• One DIN rail 
 
• Two 6-32 UNC (3.5 mm) screws 
 
• Two 3.5 mm washers 
 
INSTALLATION PROCEDURE 
 
Warning 
 
Disconnect all cables including the power cord from the unit before 
performing the mounting procedures. 
 
 To install a single unit: 
1. Using two upper holes on the bracket, fasten the DIN rail with the 6-32 UNC screws (see Figure 1), to mount the 
device on the left-side rack rail.
