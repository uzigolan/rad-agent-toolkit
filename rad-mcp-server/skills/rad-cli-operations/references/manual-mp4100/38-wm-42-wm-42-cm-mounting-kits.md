INSTALLATION 
Warning 
Disconnect all cables including the power cord from the unit before 
performing the mounting procedures. 
Installing a 17-inch device on a wall 
You will need: 
• WM-42 kit
• Drill (not provided)
• Hammer (not provided)
• Four self-tapping screws (not provided)
• 3 mm Philips screwdriver (not provided)
 
To install the unit on a wall :
1.
Mark the drilling points on the wall, using the wall-mounting bracket as a template.
2.
Drill holes at the marked points.
3.
Hammer the wall anchors into the drilled holes.
4.
Fasten the bracket to the wall using the self-tapping screws.
5.
Insert the unit into the bracket, with the front panel facing up.
6.
Fasten the unit to the side panels of the bracket using the flat-
headed screws.

Installing a 17-inch device on a wall with cable management 
You will need: 
• WM-42-CM kit
• Drill (not provided)
• Hammer (not provided)
• Four self-tapping screws (not provided)
• 3 mm Philips screwdriver (not provided)
 
To install a 17-inch device on a wall with cable management :
1.
Mark the drilling points on the wall, using the wall-mounting bracket as a template.
2.
Drill holes at the marked points.
3.
Hammer the wall anchors into the drilled holes.
4.
Attach the cable management tray to the wall mounting bracket and fasten it, using the two pan-headed screws.
5.
Fasten the bracket/cable management tray assembly to the wall using the 4-40 UNC pan-headed screws.
6.
Insert the unit into the bracket, with the front panel facing up.
7.
Fasten the unit to the side panels of the bracket using the flat-headed screws.
