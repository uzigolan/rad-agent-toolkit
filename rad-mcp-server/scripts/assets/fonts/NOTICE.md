# Font assets

IBM Plex Mono, IBM Plex Sans, and IBM Plex Sans Condensed — subset to the
Latin unicode range only, fetched from Google Fonts. Licensed under the
[SIL Open Font License 1.1](https://openfontlicense.org/). Used by
`scripts/build_results_html.py` to embed self-contained `@font-face` data
URIs in `tests/RESULTS.html` (no external font requests at view time).
